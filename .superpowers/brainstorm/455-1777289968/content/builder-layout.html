A — Full Drag-Drop Builder

Three-panel: component palette (left) → canvas (center) → properties panel (right). Drag components onto the canvas. Click to configure. Similar to Google Forms admin or Typeform builder.

Pros

  • Most powerful UX
  • Familiar to modern users
  • Visual reordering

Cons

  • Most complex to build
  • Requires drag-drop JS library

B — Click-to-Add Section Editor

No drag-drop. Each section is an accordion/card. Click "+ Add Component" buttons to append fields in sequence. Click a field to edit its properties in a modal. Reorder via up/down arrows.

Pros

  • Simpler to build
  • Familiar accordion UI
  • Works on tablets

Cons

  • Less visual
  • Up/down reorder is slower

C — Tree Nav + Section Editor (Hybrid)

Left sidebar shows Form Group → Form hierarchy as a tree. Clicking a form opens the section editor (same click-to-add approach as B). Best of both: easy navigation across forms + simple editing.

Pros

  • Clear hierarchy visible
  • Easy to jump between forms
  • Manageable build complexity

Cons

  • Sidebar takes space
  • No drag-drop reorder
