Bulk Uploader — UI Approach

Pharmacy Config page mein Bulk Uploader kaise dikhega? Teen options hain.

A

Inline Card Section (Recommended)

Config page par ek alag card section add ho "Bulk Inventory Import" title ke saath. File drop-zone, template download, aur validate/import buttons sab is card mein honge. Steps inline dikhenge (idle → file selected → validating → results).

Pros

  • Har cheez ek jagah — modal nahi kholna
  • Validation errors clearly dikhti hain
  • Config page se consistent feel

Cons

  • Config page thodi lambi ho jaayegi
B

Modal Wizard (Step-by-step)

Config page par ek "Import Inventory" button ho. Click karne par ek large modal khule jisme 3 steps hon: (1) Upload File, (2) Validation Results, (3) Confirm Import. Clean, focused experience.

Pros

  • Page clean rahti hai
  • Step-by-step guidance
  • Focus on import task

Cons

  • Modal mein zyada data (error table) dikhana awkward
  • Extra click
C

Separate Dedicated Page

Config page par sirf "Import Inventory" link ho. Yeh ek alag /configuration/pharmacy/bulk-import page par le jaaye jahan poora uploader ho — file zone, column mapping, validation table, aur import confirmation.

Pros

  • Full page space available — bari tables dikhana easy
  • Bookmarkable URL

Cons

  • Naya route, naya view — zyada kaam
  • Config page se alag feel
